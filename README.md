🏎️ SeeAuto - Araç Bakım ve Servis Takip Sistemi
SeeAuto, otomotiv servis noktalarının operasyonel süreçlerini dijitalleştirmek için PHP ve MSSQL altyapısıyla geliştirilmiş bir Veritabanı Yönetim Sistemi projesidir.  

🚀 Proje Hakkında
Bu sistem, servislerdeki karmaşık veri trafiğini (müşteri kayıtları, araç geçmişi, randevular) merkezi bir veritabanında toplayarak yönetimi kolaylaştırmayı amaçlar.  

🛠️ Kullanılan Teknolojiler
Veritabanı: Microsoft SQL Server (MSSQL)  

Backend: PHP  

Frontend: HTML5, CSS3, JavaScript  

İletişim: AJAX (Asenkron Veri Transferi)  

✨ Öne Çıkan Özellikler
Dinamik İstatistikler: Ana ekranda canlı olarak sunulan araç, müşteri ve bakım sayıları.  

Rol Bazlı Erişim: Admin ve Müşteri için özelleştirilmiş farklı yönetim panelleri.  

Asenkron Üyelik: Sayfa yenilenmeden hızlı kayıt imkanı sunan modal yapısı.  

Randevu Yönetimi: Müşterilerin online randevu oluşturabilmesi ve durum takibi.  

📦 Kurulum Notları
Projenin çalışması için SQL/ klasörü içerisindeki VTYS_ARTIK_SON_.sql dosyasının MSSQL üzerinde çalıştırılarak veritabanı şemasının oluşturulması gerekmektedir.
